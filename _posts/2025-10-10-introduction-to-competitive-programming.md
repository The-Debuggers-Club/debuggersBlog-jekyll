Learn the fundamentals of competitive coding with our workshop covering algorithms, data structures, and problem-solving techniques for platforms like Codeforces and LeetCode.

## What You'll Learn

- **Algorithm Fundamentals**: Understanding time and space complexity
- **Data Structures**: Arrays, linked lists, stacks, queues, trees, and graphs
- **Problem-Solving Techniques**: Greedy algorithms, dynamic programming, and divide-and-conquer
- **Platform Navigation**: How to effectively use Codeforces, LeetCode, and HackerRank

## Workshop Details

- **Duration**: 3 hours
- **Prerequisites**: Basic programming knowledge in any language
- **Materials**: Laptop with internet connection
- **Certification**: Certificate of participation provided

## Topics Covered

1. **Time Complexity Analysis**
   - Big O notation
   - Common time complexities
   - Space complexity considerations

2. **Essential Data Structures**
   - Arrays and strings
   - Linked lists
   - Stacks and queues
   - Trees and binary search trees
   - Graphs and graph traversal

3. **Algorithm Design Patterns**
   - Two pointers technique
   - Sliding window
   - Binary search
   - Greedy algorithms
   - Dynamic programming basics

4. **Practice Problems**
   - Hands-on coding sessions
   - Problem-solving strategies
   - Debugging techniques

## Registration

This workshop is open to all BHU students interested in competitive programming. No prior experience with competitive coding is required, but basic programming knowledge is recommended.

Join us to kickstart your competitive programming journey and connect with fellow coding enthusiasts!
